Write a function that takes two integers and returns their sum.

## Requirements

Implement a function with the following signature:

- `int add(int a, int b)` - Return the sum of the two input integers

## Examples

```java
add(2, 3);     // returns 5
add(-1, 1);    // returns 0
add(0, 0);     // returns 0
add(-5, -3);   // returns -8
add(100, 200); // returns 300
```
